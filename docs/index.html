DotBox UI Components

A modern, lightweight UI component library for building beautiful web applications.

Buttons

Flexible button component with multiple variants and states.

const button = new DotBox.Button({
    text: 'Click Me',
    variant: 'primary',
    size: 'medium',
    onClick: () => console.log('Clicked!')
});

document.body.appendChild(button.getElement());

Modal Dialogs

Modal dialogs for displaying content in an overlay.

const modal = new DotBox.ModalDialog('demo-modal', {
    closeOnOverlayClick: true,
    closeOnEsc: true
});

modal.setTitle('Demo Modal')
    .setBody('
This is a demo modal dialog.
')
    .addFooterButton('Close', 'action-btn', () => modal.close());

const openModalBtn = new DotBox.Button({
    text: 'Open Modal',
    onClick: () => modal.show()
});
document.getElementById('modal-demo').appendChild(openModalBtn.getElement());

Tab Views

Tabbed interface for organizing content.

const tabView = new DotBox.TabView({
    tabs: [
        { id: 'tab1', label: 'Tab 1', content: 'Content 1' },
        { id: 'tab2', label: 'Tab 2', content: 'Content 2' }
    ]
});

document.body.appendChild(tabView.getElement());

Text Boxes

Input fields with various states and validations.

const textBox = new DotBox.TextBox({
    label: 'Username',
    placeholder: 'Enter your username',
    helpText: 'This will be your display name',
    onChange: (e) => console.log('Text changed:', e.target.value)
});

document.getElementById('textbox-demo').appendChild(textBox.getContainer());
